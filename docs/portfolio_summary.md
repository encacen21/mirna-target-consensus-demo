# Portfolio Summary

## Project

**miRNA Target Consensus Demo**

## Repository

https://github.com/encacen21/mirna-target-consensus-demo

## Summary

This project is a synthetic Python bioinformatics workflow that demonstrates how outputs from multiple miRNA target-prediction tools can be normalized, integrated, summarized, and converted into consensus-based target tables.

The workflow uses only mock data and is designed as a safe public portfolio example. It does not contain unpublished research data, real thesis results, real biological candidates, real sample names, or experimental validation.

## Skills Demonstrated

- Python scripting for bioinformatics workflows
- pandas-based table processing
- normalization of heterogeneous prediction outputs
- multi-tool evidence integration
- consensus scoring
- generation of target-gene lists by miRNA
- summary reporting
- reproducible workflow execution
- GitHub-ready documentation
- safe public release using synthetic data

## Workflow Components

The repository includes scripts for:

1. Normalizing mock predictions into a standardized long-format table.
2. Building a consensus table from multiple prediction tools.
3. Exporting target-gene lists per miRNA.
4. Summarizing prediction support by tool and consensus level.
5. Running the full demo workflow with a single command.

## Suggested Upwork Description

Developed a synthetic Python workflow that integrates mock outputs from multiple miRNA target-prediction tools into standardized, consensus-based target tables. The project demonstrates data normalization, multi-tool evidence integration, consensus scoring, gene-list generation, summary reporting, and reproducible workflow execution using Python and pandas.

The repository uses only synthetic mock data and clearly documents its scientific limitations, making it suitable as a public portfolio example for RNA/miRNA bioinformatics workflows.
